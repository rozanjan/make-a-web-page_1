LESSON ONE: Basics of Web and HTML


The World Wide Web
The World Wide Web is a collection of document pages stored on computers called servers. Individual computers can access Web pages through the internet using http. Http is the address that tell the server how to locate the web page. The Udacity Video provides a nice introduction to the web.
HTML
HTML is the coding language that most of the pages on the web are written in. The full name for HTML is hypertext markup language.
Tags are shortcuts that tell the browser how to display a page. Html is written by having an opening tag, then the contents and then the closing tag. Tags can be inline such as a, span, br, ing, stong or block div, p, form . Inline means the code works within the line and block means it draws an invisible block around the text.
Stupid Computers
Another main point to remember is that computers take everything literal. A little typo can mess up the entire page.

LESSON TWO: Creating Structured Documents


Web Development
Web pages can consist of three programming languages. HTML controls the structure of the page. CSS controls the style of the page. Java Script controls the interactive components.
Tree Structure
Html is written in a tree like structure. Each box of code branches out into other branches. This tree like structure is also called DOM (Document Object Model). Html is written in a wave of indentation on the left side of a document. An element is considered deepely nested if it is more indented inside it's box.

Lesson 3: CSS STYLE


Avoid Repetition
Avoiding repetition is important to avoid errors. If you are writing the same code over and over it is very easy to make errors. If then you want to change the code, every line of repetition must be changed to avoid problems. Avoiding repetition is also important to maintain consistency when you are adding style elements.
CSS Cascading Style Sheets (CSS) is a way of adding style to a web page. Elements on a page are named and then the style such a font, color, etc can be applied to the elements.
